Trigon Rules

Trigon is another game variant. The rules a similar to game variant Classic but is uses a differently shaped board and set of pieces. Each color uses 22 pieces that are shaped like the polyiamonds up to size six. (A polyiamond is a figure built by a number of equilateral triangles connected along the edges.)

The 22 pieces:

The board also consists of triangles and is shaped like a hexagon with nine triangles building an edge.

The board with the starting
fields marked with grey dots:

There are six starting points on the board, each located four rows from the middle of an edge. The starting points are not colored and the players may freely choose a starting point for the first piece of a color.

Trigon Rules for Two Players

Like game variant Classic, Trigon can be played with two players by having one player play Blue and Red and the other player Yellow and Green.

Pentobi User Manual

Pentobi is a computer program that plays the board game Blokus. In this game, four players place pieces similar to the pieces of the computer game Tetris on a 20×20 board. Pentobi also supports two-player game variants. The homepage of Pentobi is at http://pentobi.sourceforge.net/
